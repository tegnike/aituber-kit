# 商標使用條款

## 概述

本文檔規定了在[aituber-kit](https://github.com/tegnike/aituber-kit)儲存庫中使用的商標的使用條款。

## 允許使用

- 與本儲存庫相關的非商業目的使用
- 在介紹本儲存庫時使用商標
- 用於連結目的（僅限於與本儲存庫相關的網站）

## 禁止使用

儲存庫的所有者有權在不事先通知的情況下變更此使用協議。用戶有責任定期檢查協議並了解變更。

## 聯絡方式
